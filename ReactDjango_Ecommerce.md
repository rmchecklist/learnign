# Prerequisite:

1. Visual Studio code
    Extensions:
      Prettier
      Auto Reanme tag
      Bracket pair colorizer
      ES7
      
      
2. Create React app - FrontEnd

```
npx create-react-app frontend
```

3. Start the app

```
cd frontend
npm start
```
